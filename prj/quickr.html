기술설명

퀵 업체 서버와 연동하여 실제 사용자와 퀵 배달원들이 사용하는 어플리케이션. 실제 사용자가 본인의 위치와 배달 위치를 지정하면 배달원이 방문하여 배달하는 서비스로 하나의 SDK 를 만들어놓고 패키지만 다르게 해서 빌드를 함.
빌드의 경우 Shell Script를 사용하여 Jenkins로 구축을 하였으며, 필요한 정보들만 입력하면 자동으로 빌드가 되어 apk 파일을 지정한 경로에 저장하게끔 함.
